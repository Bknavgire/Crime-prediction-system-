# Crime Rate Prediction Project

## Project Overview
This project is a simple Machine Learning based Crime Rate Prediction system developed using Python, Flask, and Linear Regression.

The system predicts the crime rate based on the year entered by the user.

---

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-Learn
- HTML
- CSS
